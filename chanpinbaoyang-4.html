多功能咬口机的基础参数及特点

2013-02-27 admin

  • 多功能咬口机的特性：
         
    1. 采用钢板焊接布局，振动时解除应力，具备精良的刚性和不乱性；
      2.    
    2. 呆板加工板材概况平坦、滑腻，尺寸同等，形状雅观；
      3.    
    3. 呆板采纳机电传动，活动时完成滚压成角，同步靠得住，活动安稳；
      4.    
    4. 极大地加重了传统手工操纵的休息强度，进步了休息出产率。
    •    
  • 多功能咬口机的技能参数：
         
    1. 加工板材厚度：0.5-1.6mm；
      2.    
    2. 咬口形状：结合角、单平口、直角、弯头；
      3.    
    3. 形状尺寸：1200*700*1050mm；
      4.    
    4. 电念头：（功率）1.5kw-6极；（转速）940r/min；
    5. 分量：300kg；
    •    
  • 多功能咬口机主要由机架部分、结合角咬口机芯、弯管咬口机芯和传动体系等构成。对于各个部门其分别的特点是：
         
    1. 机架部分：由型材和板材焊接而成，底部装有四只脚轮，具备精良的刚性。
      2.    
    2. 结合角、直角咬口机芯：由发动部位和下面的滚轮组和墙板构成，具备精良的刚性和不乱平坦的成形性；直角咬口机芯由滚轮组、墙板和上面的发动部位构成咬口成型雅观耐用。
      3.    
    3. 弯管咬口机芯：由底座、底座盖和压板构成，布局谨严，不乱性强，且加工形状雅观，机能精良；另加一套单平口咬口形状模具，组成四种功效的多用咬口机。
